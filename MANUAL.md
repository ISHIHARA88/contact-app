# お問い合わせ管理アプリ 操作マニュアル

---

## 1. アプリの概要

本アプリは、ホームページに設置した **お問い合わせフォーム** への回答を LINE で即時通知し、専用の **管理画面** で一括管理できるツールです。

**主な機能**

- お客様がフォームから送信 → 担当者の LINE に通知が届く
- 管理画面にログインしてお問い合わせ内容を一覧・確認
- 対応済みのものを「完了」に更新、不要なものを削除
- スマートフォンからでも見やすいカード形式で表示

---

## 2. お問い合わせフォームの使い方

ホームページ（`https://（サイトURL）/`）にアクセスすると、フォームが表示されます。

| 入力項目 | 説明 |
|---|---|
| お名前 | お客様の氏名（必須） |
| メールアドレス | 返信先となるメールアドレス（必須） |
| お問い合わせ内容 | ご質問・ご要望など（必須） |

すべての項目を入力して「送信する」ボタンを押すと、管理者の LINE に通知が届き、管理画面にも内容が記録されます。

---

## 3. 管理画面へのログイン方法

### アクセス先

```
https://（サイトURL）/admin
```

未ログインの場合は自動的にログイン画面へ移動します。

### ログイン手順

1. **メールアドレス** と **パスワード** を入力する
2. 「ログイン」ボタンを押す
3. 認証が通ると管理画面が表示される

> ログイン情報は別途お伝えします。第三者に共有しないようご注意ください。

### ログアウト

管理画面右上の「ログアウト」ボタンを押すと安全にログアウトできます。

---

## 4. お問い合わせ一覧の見方・操作方法

### 一覧の表示

ログイン後、受信したお問い合わせが新しい順に表示されます。

- **PC・タブレット**：テーブル形式で表示。行をクリックすると詳細がポップアップで表示されます。
- **スマートフォン**：1件ずつカード形式で表示。カードをタップすると詳細が表示されます。

### ステータスの見方

| ステータス | 意味 |
|---|---|
| 未対応（黄色） | まだ対応が完了していないお問い合わせ |
| 完了（緑色） | 対応済みのお問い合わせ |

### 操作方法

詳細ポップアップ内から以下の操作ができます。

**完了にする**
対応が終わったら「完了にする」ボタンを押すと、ステータスが「完了」に更新されます。

**削除する**
「削除」ボタンを押すと確認メッセージが出た後、一覧から削除されます。削除したデータは復元できませんので、ご注意ください。

## 5. LINE 通知の設定方法

フォームが送信されると、LINE Messaging API を通じて担当者に通知が届きます。初期設定が必要な場合は、以下の手順で行ってください。

### 5-1. LINE Developers でアクセストークンを発行する

1. [LINE Developers コンソール](https://developers.line.biz/console/) にアクセスしてログイン
2. 対象の **Messaging API チャネル** を選択
3. 「Messaging API 設定」タブを開く
4. 画面下部の「チャネルアクセストークン（長期）」欄の「発行」ボタンを押す
5. 表示されたトークンをコピーして控える

### 5-2. ユーザー ID を確認する（個人への通知に必要）

1. 同じ Messaging API 設定ページの「あなたのユーザー ID」欄をコピーして控える

### 5-3. Vercel の環境変数に登録する

1. [Vercel ダッシュボード](https://vercel.com/) にログイン
2. 対象のプロジェクトを選択 → 「Settings」→「Environment Variables」を開く
3. 以下の変数を追加する

| 変数名 | 値 |
|---|---|
| `LINE_CHANNEL_ACCESS_TOKEN` | 5-1 でコピーしたアクセストークン |
| `LINE_USER_ID` | 5-2 でコピーしたユーザー ID |

4. 追加後、「Deployments」から最新のデプロイを **Redeploy** して設定を反映させる

### 5-4. 通知用アカウントを友だち登録する

LINE Messaging API の通知を受け取るには、通知元となる **LINE 公式アカウント（Bot）** を友だち登録しておく必要があります。

1. LINE Developers コンソールの「Messaging API 設定」タブを開く
2. 「Bot 情報」欄にある QR コードを LINE で読み取る
3. 友だち登録が完了すると、フォーム送信時に通知が届くようになる

## 6. パスワードの変更方法

### 方法 A：ログイン中に変更する（通常の変更）

1. 管理画面右上の「パスワード変更」ボタンを押す
2. 新しいパスワードを入力（6文字以上）
3. 確認用パスワードを再入力
4. 「パスワードを変更する」ボタンを押す
5. 「パスワードを変更しました」と表示されれば完了

### 方法 B：パスワードを忘れた場合（メールでリセット）

1. ログイン画面（`/login`）を開く
2. 画面下部の「パスワードを忘れた方はこちら」をタップ
3. 登録済みのメールアドレスを入力して「リセットメールを送る」を押す
4. 届いたメール内のリンクをタップする
5. 新しいパスワードを入力して「パスワードを変更する」を押す
6. ログイン画面へ自動的に戻るので、新しいパスワードでログインする

> リセットメールのリンクには有効期限があります（通常24時間）。期限切れの場合は、再度リセット申請を行ってください。
